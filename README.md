# Project Phoenix

> **Building again. One meaningful commit at a time.**

Project Phoenix is an open journey of rebuilding.

It documents the process of creating real-world products, developing AI capabilities, growing Kopi Tjap Spoor, and building a sustainable career through continuous learning and practical execution.

This repository is not a collection of finished work.

It is a record of how meaningful work gets built.

---

# Why Project Phoenix?

Project Phoenix exists because building is more valuable than waiting.

Instead of chasing perfect conditions, the goal is to make the next meaningful commit, learn from reality, and continuously improve.

The project combines entrepreneurship, software engineering, artificial intelligence, and documentation into one long-term builder journey.
